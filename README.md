# Cadastro-de-Produtos

## Objetivo 
O cadastro de produtos utiliza o powershell, aonde ele consegue registar o seu produto, o número dele, o valor dele, a quantia presente no estoque e consegue mudar essa quantia quando um produto é vendido. Também tem a capacidade de excluir o produto e a opção de fechar o sistema.
## O que foi desenvolvido:
Habilidades novas com a programação em python, melhorar a minha lógica de programação.
